Шаблон: Абстрактная фабрика (Abstract factory)
======================================

Цель:

Создание интерфейса, для создания множества взаимосвязанных
или взаимозависимых обьектов, без жесткой привязки к
конкретным классам

Позволяет создавать семейства связанных обьектов, не привязываясь
к конкретным классам создаваемых объектов

========================================

Для чего используется:

Для создания множества взаимосвязанных объектов

==========================================

Примеры использования:

- система не должна зависить от метода создания, компоновки и
  представления входящих в нее объектов
- входящие взаимосвязанные объекты должны использоваться вместе
- система должна конфигурироваться одним из множеств объектов,
  из которых она состоит
- нам необходимо предоставить множество обьектов, раскрывая только
  их интерфейсы но не реализации.
